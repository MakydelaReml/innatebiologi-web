# Checklist de continuidad y aprobacion — Clase 1 piloto

## Antes de producir un plano

- [ ] El plano tiene ID.
- [ ] Tiene duracion provisional.
- [ ] Tiene funcion narrativa.
- [ ] Tiene imagen inicial e imagen final.
- [ ] Tiene movimiento de camara definido.
- [ ] Tiene movimiento secundario definido.
- [ ] Tiene metodo tecnico asignado.
- [ ] Tiene riesgos anotados.
- [ ] Tiene criterio de aprobacion.

## Continuidad visual

- [ ] Hora del dia: preamanecer hasta primer sol.
- [ ] Luz coral solo en horizonte antes del sol.
- [ ] Oro biologico reservado a pulso/conduccion.
- [ ] El lago refleja una luz adelantada, sin parecer portal.
- [ ] El arbol no aparece como logo plano.
- [ ] La puerta no es literal.
- [ ] No hay elementos modernos salvo la pantalla del Maestro.
- [ ] No aparecen marcas ni textos accidentales.

## Continuidad de camara

- [ ] P1.2: macro lento.
- [ ] P1.3: FPV sereno, no persecucion.
- [ ] P1.4: descenso y desvio antes del umbral.
- [ ] P1.5: altura cintura, rostros no completos.
- [ ] P1.6: 85 mm fijo.
- [ ] P1.7: revelacion luminosa, no videoclip.

## Continuidad de personajes

- [ ] Tomas: se identifica por botas/peso/terreno.
- [ ] Lucia: se identifica por mano y rama.
- [ ] Ines: puente, paso contenido.
- [ ] Nico: charco/reflejo/imaginacion.
- [ ] Amina: semilla/cuidado.
- [ ] Elena: pausa/respiracion.
- [ ] Maestro: mano/presencia/objetos, sin rostro revelado.

## Sonido

- [ ] Respiracion humana no dramatizada.
- [ ] Latido grave, no terror.
- [ ] Tierra y raiz con textura organica.
- [ ] Viento de montana suave.
- [ ] Agua amplia.
- [ ] Pasos y tela en P1.5.
- [ ] Objeto apoyado en raiz en P1.6.
- [ ] Motivo de memoria entra casi inaudible.

## Aprobacion por fases

### Frame maestro

- [ ] Composicion aprobada.
- [ ] Paleta aprobada.
- [ ] Nivel de fantasia controlado.
- [ ] No hay artefactos graves.

### Movimiento

- [ ] Camara coherente.
- [ ] Parallax natural.
- [ ] Movimiento secundario ayuda y no distrae.
- [ ] No hay deformacion intolerable.

### Montaje

- [ ] Ritmo respira.
- [ ] Transiciones se sienten inevitables.
- [ ] El titulo entra tarde.
- [ ] El piloto deja ganas de entrar en la clase.

## Regla final de aprobacion

Si el plano es bello pero no sirve a la pregunta de la clase, no esta aprobado.
