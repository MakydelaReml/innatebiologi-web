# Requisitos de model sheets — Clase 1

## Entregable por cada personaje humano

Formato recomendado: lamina horizontal 16:9 o vertical A3.

Contenido minimo:

| Bloque | Requisito |
| --- | --- |
| Turnaround | frontal, 3/4, perfil, espalda |
| Cuerpo | cuerpo completo con proporcion clara |
| Rostro | neutro + 5 expresiones |
| Poses | sentado escuchando, caminando, reaccionando |
| Manos | 3 detalles de mano |
| Vestuario | materiales, capas, desgaste, accesorios |
| Paleta | 6-10 colores |
| Escala | junto a Maestro y/o arbol |
| Prompt base | descripcion bloqueada para IA |
| Negativos | cosas que nunca debe generar |

## Entregable por cada entidad biologica

Objetivo: convertir personajes demasiado infantiles en presencias biologicas adultas.

Contenido minimo:

| Bloque | Requisito |
| --- | --- |
| Forma base | silueta principal adulta |
| Variaciones | 3 estados o intensidades |
| Materialidad | agua, mineral, grasa, fibra, acido, etc. |
| Funcion narrativa | para que aparece en clase |
| Movimiento | como se desplaza o expresa |
| Paleta | colores sobrios |
| Prohibido | ojos enormes, sonrisa infantil, cuerpo de juguete |

## Entregable por escenario

Contenido minimo:

| Bloque | Requisito |
| --- | --- |
| Master layout | vista general cinematografica |
| Top-down | mapa funcional para camaras |
| Zonas | lago, arbol, circulo, senderos, puente |
| Camaras | posiciones de planos importantes |
| Luz | preamanecer, primer sol, mañana, tarde |
| Texturas | corteza, agua, piedra, hierba, tierra |
| Escala | figura humana de referencia |
| Reglas | que cambia y que no cambia |
