# Workbook Clase 4

## Aprender a observar el Estado Conductivo

### Proposito

Entrenar observacion sin convertirla en diagnostico.

### Idea central

El Estado Conductivo no es una etiqueta sobre una persona. Es una forma de observar condiciones relacionadas con conductividad biologica.

### El arbol no responde deprisa

Un arbol no explica cada hoja por separado. Muestra tendencias: luz, agua, estacion, suelo, dano, reparacion.

Escribe:

**Que tendencia podria estar intentando ver demasiado rapido?**

### Cierre

Observar no es concluir. Observar es crear las condiciones para comprender mejor.

## Mapa del alumno

| Aprendizaje | Como se practica |
| --- | --- |
| Estado Conductivo no es identidad | Hablo de condiciones, no de personas |
| Observar no es interpretar | Registro antes de explicar |
| Los patrones necesitan tiempo | Miro tendencias, no instantes |
| La memoria del organismo aparece en ritmos | Atiendo continuidad y contexto |

## Ejercicio 3: lenguaje prudente

Reescribe frases para que sean observacionales.

| Frase demasiado cerrada | Version prudente |
| --- | --- |
| Mi cuerpo esta mal | Estoy observando una senal que necesita contexto |
| Esto significa que... | Una posibilidad a observar seria... |
| Soy asi | En este momento aparecen estas condiciones... |
