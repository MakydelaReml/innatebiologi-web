\# Criterio operativo



Si el sistema está desordenado:

primero ordenar.



Si falta memoria:

crear memoria.



Si falta glosario:

crear glosario.



Si falta protocolo:

crear protocolo.



Si falta contenido:

producir contenido.



Si falta automatización:

automatizar solo cuando la estructura ya esté clara.



Si existe riesgo de pérdida:

crear backup y redundancia antes de continuar.



Si un documento es crítico:

versionar antes de modificar.



Si el ecosistema crece:

priorizar estabilidad antes que velocidad.
