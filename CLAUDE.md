# Instrucciones de proyecto para Claude Code

Este repositorio comparte memoria y canon con Codex. Antes de modificar contenido, arquitectura, clases, guiones, marca o experiencia del alumno, lee:

1. `AGENTS.md`
2. `INNATEBIOLOGI_AI/DOCUMENTOS_MAESTROS/CANON_INNATEBIOLOGI_V1.md`
3. `INNATEBIOLOGI_AI/DOCUMENTOS_MAESTROS/ARQUITECTURA_REPOSITORIOS_V1.md`
4. `INNATEBIOLOGI_AI/INDICE_MAESTRO_V1.md`

Respeta la autoridad documental indicada en esos archivos. No crees una segunda versión del canon ni conviertas resultados de `output/`, `tmp_character_sheets/` o `graphify-out/` en fuentes oficiales.

Si trabajas desde `innatebiologi-academia`, el código de la academia pertenece a ese repositorio, pero cualquier cambio de canon, voz, ciencia, marca o decisión maestra debe integrarse primero en la fuente común definida por `ARQUITECTURA_REPOSITORIOS_V1.md`.

El fundador conserva la decisión final sobre el alma del proyecto. Formula las cautelas científicas y legales sin borrar el significado del método ni sustituir el modelo educativo propio por otro proyecto distinto.
